# 💼 BTG Financeira

Sistema web desenvolvido em **ASP.NET (.NET 8)** com foco em simulação e apresentação de serviços financeiros, como crédito consignado CLT.

---

## 🚀 Sobre o Projeto

O **BTG Financeira** é uma aplicação web moderna que simula um ambiente de serviços financeiros, com foco em:

* 💳 Empréstimo consignado CLT
* 📊 Simulação de crédito
* 👥 Interface amigável para usuários
* 🎨 Layout moderno e responsivo

---

## 🛠️ Tecnologias Utilizadas

* ⚙️ ASP.NET Core (.NET 8)
* 🎨 HTML, CSS, JavaScript
* 🧩 Bootstrap
* 💾 C#

## ▶️ Como Executar o Projeto

1. Clone o repositório:

```bash
git clone https://github.com/gabrielpereira21/BTG-Financeira.git
```

2. Acesse a pasta:

```bash
cd BTG-Financeira
```

3. Execute o projeto:

```bash
dotnet run
```

4. Acesse no navegador:

```
https://localhost:5001
```
